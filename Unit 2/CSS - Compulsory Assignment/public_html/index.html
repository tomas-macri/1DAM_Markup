HTML

HTML, stands for Hyper Text Markup Language (Lenguaje de Marcado de Hipertexto), is the most used markup language for the development of web pages. It is used to describe the srtucture and the content in a text format, as to complement the text with objects like images. HTML is written in "tags", surrounded by angle brackets (<,>). HTML can also describe, up to a certain point, the look of a document, and may include a script (for example JavaScript), which can affect the behaivour of web browsers and other HTML proccesors.
HMTL is also used to refer to the content type of MIME text/html oe even wider like a generic definition for the HTML, either in a XML form (like XHTML 1.0 and (further) or in a descending way directly from SGML (like HTML 4.01 and previous).
